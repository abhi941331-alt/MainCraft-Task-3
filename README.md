# Personal Portfolio Website

This project now includes Web Development Internship Task 3 features on Abhinav's personal portfolio website.

Live Demo - https://abhi941331-alt.github.io/MainCraft-Task-3/

## Included Requirements

- Hero section based on the PDF starter code
- About Me, Projects, and Contact sections
- CSS hover animation for buttons
- Smooth CSS-only transitions and reveal animations
- Responsive layout with a `max-width: 768px` media query
- Google Fonts usage
- Dynamic navbar style on scroll
- Active navigation link highlighting
- Contact form validation with inline error messages
- Dark mode toggle with `localStorage` preference saving
- GitHub Pages-ready static files

## Files

- `index.html` - page structure
- `styles.css` - layout, animations, responsive design
- `script.js` - dynamic navbar, form validation, and dark mode toggle

## Run Locally

Open `index.html` in a browser.

## Deploy With GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and this `README.md`.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and `/root` folder.
7. Save and wait for GitHub to publish the live link.
